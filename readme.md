1️⃣ What is the difference between var, let, and const?
Answer:
var,let and const are used in javascript for variable declaration but their behaviour are not same.
var is a function scope but let and const is a block scope,var can be redeclare but let and const can't redeclare,var and let can be reassign but const can't be reassigned.



2️⃣ What is the spread operator (...)?
Answer:
spread operator(...) used in javascript to expand elements of an array or object.

example:
const nums=[1,2,3]
console.log(...nums)
